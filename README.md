## 🧪 實驗室筆記 (Learning Logic)

玩家可以透過不同的組合，解鎖愛迪生的各種趣味評價：

| 組合配方 (電池 + 燈泡) | 特調名稱 | 科學原理 | 愛迪生的反應 |
| :--- | :--- | :--- | :--- |
| **串聯 + 並聯** | 超級閃亮特調 | 最大電壓 + 獨立路徑 | 「電力強到可以看到外星人！」 |
| **並聯 + 串聯** | 微弱螢火蟲特調 | 低電壓 + 分壓變暗 | 「電力太弱，連牙刷都動不了。」 |
| **串聯 + 串聯** | 普通亮度特調 | 高電壓 + 分壓抵銷 | 「四平八穩，雖然不刺激但很安全。」 |
| **並聯 + 並聯** | 持久續航特調 | 低電壓 + 能量分散 | 「可以連續跳舞三天三夜不休息！」 |

## ✨ 遊戲特色

* **趣味角色互動**：以科學家愛迪生為主角，增加學習的帶入感。
* **視覺化回饋**：根據特調結果切換 Emoji 與背景顏色（如：成功後的綠色、短路後的淡紅色）。
* **純前端運作**：經過優化，無需任何後端伺服器即可在 GitHub Pages 上完美執行。
* **適合行動裝置**：支援響應式設計，在平板或手機上也能輕鬆實驗。

## 🛠️ 技術細節

* **語言**: HTML5, CSS3, JavaScript (ES6+)。
* **邏輯架構**: 採用單一 HTML 檔案架構，將原本的 Google Apps Script 後端邏輯成功移植至前端 JavaScript 函式中。

## 📖 如何使用

1.  點擊「開始特調」。
2.  選擇電池接法（串聯或並聯）。
3.  選擇燈泡接法（串聯或並聯）。
4.  觀察愛迪生的表情與評價，學習電路知識！
